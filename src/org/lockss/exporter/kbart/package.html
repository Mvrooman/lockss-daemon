Exporter converting metadata to the KBART format and exporting in CSV or HTML. Exports can be created for all available titles, configured titles or preserved titles. The contents of the latter are decided based upon a value threshold for the org.lockss.daemon.AuHealthMetric.

The KBART recommendations define an ordinal list of fields which must all appear in the output. Given that our data is incomplete and entire fields can be empty, and the fact in some applications a pared-down or restructured version of the data is more useful, certain methods of customising the output have been included. Though these do not produce a strict KBART output format, the contents of the fields are KBART compatible.

We have an entry in the KBART registry which should be kept up to date.
